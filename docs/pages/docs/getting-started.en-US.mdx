# Getting Started

## Installation

Inside your React project directory, run the following:

```plaintext
yarn add wagmi ethers
```

Or with npm:

```plaintext
npm install wagmi ethers
```

## Quick Start

Wrap your app with the [`Provider`](TODO) component.

```tsx
import { createClient, Provider } from 'wagmi'

const App = () => (
  <Provider>
    <YourRoutes />
  </Provider>
)
```

<Callout>
  You can specify multiple configuration options, such as: connectors (i.e.
  WalletConnect), auto-connection, websockets, etc, by passing a "wagmi client"
  instance to the `Provider` via the `client` prop. [Read more](TODO)
</Callout>

Use hooks! Every component inside the Provider is set up with the default InjectedConnector for connecting wallets and ethers.js [Default Provider](https://docs.ethers.io/v5/api/providers/#providers-getDefaultProvider) for fetching data.

```tsx
import { useAccount } from 'wagmi'

function AccountDetails() {
  const { data, isLoading, disconnect } = useAccount({
    ens: true,
  })

  return ...
}
```
